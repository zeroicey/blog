# 你好，我是雪宁韵

GitHub 上我使用的英文名是 `zeroicey`。目前我把这个小站当作自己的记录空间，用来放一些技术思考、项目探索，以及那些值得慢慢整理下来的想法。

我的一句座右铭是：

> 去码头整点薯条

## 我在做什么

从 GitHub 上公开能看到的内容来看，我最近关注的方向比较杂，但也很有个人风格：

- 个人生活记录与健康相关产品
- 偏安静、克制风格的效率工具
- Neovim 插件与开发体验优化
- 一些更偏底层或图形方向的小项目

当前公开项目里，比较能代表这些兴趣的有：

- [LifeSync-Health-Ultra-Web](https://github.com/zeroicey/LifeSync-Health-Ultra-Web)：围绕身心协同健康管理的 Web 项目
- [serenique](https://github.com/zeroicey/serenique)：强调清晰与平静感的个人生活追踪系统
- [bufswi.nvim](https://github.com/zeroicey/bufswi.nvim)：一个轻量的 Neovim buffer 切换插件
- [minecraft](https://github.com/zeroicey/minecraft)：使用 C++ 和 raylib 的小型 Minecraft clone

## 我关心的东西

我对“接口”“抽象”“可替换性”这类编程里的基础概念一直很有兴趣。比起只追求把功能先堆出来，我更在意代码是否清楚、是否容易演化、以及不同实现之间能不能平滑切换。

这也是为什么这个博客里会出现一些偏概念、偏思考型的文章。它们不一定都很宏大，但会尽量诚实地记录我当下的理解。
